<!-- Adapted from https://github.com/mattpocock/skills (skills/productivity/grill-me) for second-brain / knowledge-work use. -->

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the decision tree, resolving dependencies between decisions one by one. For each question, provide your recommended answer.

Ask the questions one at a time.

If a question can be answered by reading the vault (context files in 00-Context/, the relevant project or area files, recent daily notes), read the vault instead of asking. Folder names here are the template defaults — resolve the actual ones from the "Folder roles" table in CLAUDE.md's Vault structure section.

When the grilling is done:
1. Summarize the resolved decisions and any remaining open points.
2. Offer to record the outcome in the right place — usually the relevant project file in 02-Projects/ or area file in 03-Areas/, with open points as todos (which, per vault rules, also go into today's daily note, linked).
